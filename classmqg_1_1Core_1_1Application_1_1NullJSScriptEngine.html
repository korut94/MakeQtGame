Detailed Description

The Application::NullJSScriptEngine class defines a null service for scripting task.

In way to avoid temporary states where the script engine service in the Application class is no setted (that should involve in a dereferentation of a null pointer), the NullJSScriptEngine provides a null service where in any method nothing is done.

Note
This class not should be use in the application directly, since the Application class worries about the instantiation of a null service when requirement. For example:
Application::provide(nullptr); // Application will use a NullJSScriptEngine
